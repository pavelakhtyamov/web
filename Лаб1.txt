# Для выполнения работы была выбрана страница сайта elecomp.ru с поиском (elecomp.ru/search)
# Программные инструменты: "инструменты разработчика" в браузере Microsoft Edge и утилита wget.exe.
# Каталог расположения утилиты Wget в переменные среды Windows в системной переменной Path для возможности запуска утилиты из любого месторасположения в командной строке.
# С помощью браузера был рассмотрен запрос, отсылаемый сайту для поиска нужной информации.
# Установлено, что сайту передается всего один параметр: searchstring.
# Затем в командной строке Windows был выполнен следующий скрипт:

C:\Windows\system32>wget https://elecomp.ru/search/ --post-data=searchstring: sony --no-check-certificate

# Данным скриптом был произведен поиск на указанном сайте и создан файл index.html с результатом поиска.
